# Alpha APK Store

A complete app store platform for Android APKs.

## Features
- User authentication (Register/Login)
- Browse apps by category
- Search apps
- Download APKs
- Favorite apps
- Rate and review
- Dark/Light mode
- Responsive design

## Tech Stack
- **Backend**: Node.js, Express, MongoDB, JWT
- **Frontend**: HTML, CSS, JavaScript
- **Hosting**: Render.com
